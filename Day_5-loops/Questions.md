Here are 5 beginner‑friendly loop questions you can practice in C++:

### 🔄 Loop Practice Questions
1. **Print Numbers 1 to 10**  
   Write a program using a `for` loop to print numbers from 1 to 10.

2. **Sum of First N Natural Numbers**  
   Take an integer `N` as input and use a `while` loop to calculate the sum of numbers from 1 to `N`.

3. **Multiplication Table**  
   Use a `for` loop to print the multiplication table of a number entered by the user (e.g., 5 × 1 … 5 × 10).

4. **Reverse Counting**  
   Write a program using a `while` loop to print numbers from 50 down to 1.

5. **Factorial of a Number**  
   Use a `for` loop to calculate the factorial of a given number (e.g., 5! = 120).

Here are three beginner-friendly practice problems based on **Input/Output** and the task of printing `"Hello World"`:

---

### 📝 Practice Problems

1. **Print Your Name**
   - Write a program that prints your name on the screen.
   - Example Output:  
     ```
     My name is Shreyan
     ```

2. **Print a Quote**
   - Write a program that prints your favorite quote or saying.
   - Example Output:  
     ```
     "Knowledge is power."
     ```

3. **Print Multiple Lines**
   - Write a program that prints three different lines of text, one after another.
   - Example Output:  
     ```
     Hello World
     Welcome to Programming
     Let's learn C++
     ```

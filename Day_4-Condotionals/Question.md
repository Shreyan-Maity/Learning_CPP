Here are 5 practice questions on **conditionals in C++** that you can try solving:

### 📝 Questions
1. **Even or Odd Check**  
   Write a program that takes an integer input and prints whether it is even or odd using an `if-else` statement.

2. **Grade Evaluation**  
   Given a student's score (0–100), print the grade based on the following rules:  
   - 90–100 → "A"  
   - 75–89 → "B"  
   - 50–74 → "C"  
   - Below 50 → "Fail"

3. **Leap Year Finder**  
   Write a program that checks if a given year is a leap year.  
   (Hint: A year is a leap year if it is divisible by 4 but not by 100, OR divisible by 400.)

4. **Maximum of Three Numbers**  
   Take three integers as input and print the largest number using nested `if` statements.

5. **Simple Calculator**  
   Write a program that takes two numbers and an operator (`+`, `-`, `*`, `/`) as input. Use `if-else` statements to perform the correct operation and print the result.
